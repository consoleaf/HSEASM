* Условие

#+BEGIN_QUOTE
Разработать программу, которая вводит одномерный массив =A[N]=,
формирует из элементов массива =A= новый массив =B= по правилам,
указанным в таблице, и выводит его.

Вариант: 8
Массив =B= из... элементов =A= в обратном порядке
#+END_QUOTE

* Код основной программы
#+INCLUDE: main.asm src asm

* Сборка (Linux x64)

#+BEGIN_SRC bash
nasm -f elf main.asm
ld -m elf_i386 main.o -o main
#+END_SRC

* Запуск

#+BEGIN_SRC bash
./main
#+END_SRC

* Примеры работы

** Обычное использование

#+BEGIN_EXAMPLE
❯ ./main
Введите длину массива
>> 5
Введите числа массива
>> 1
>> 2
>> 3
>> 4
>> 5
Введённый массив: 1 2 3 4 5
Новый массив:     5 4 3 2 1
#+END_EXAMPLE

** Пустой массив

#+BEGIN_EXAMPLE
❯ ./main
Введите длину массива
>> 0
Пустой массив, нечего делать.
#+END_EXAMPLE

** Один элемент в массиве

#+BEGIN_EXAMPLE
